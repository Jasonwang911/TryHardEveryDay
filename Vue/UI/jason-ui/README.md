### 单元测试
测试框架：   
1. karma   可以在浏览器中执行测试用例，为前端自动化测试提供跨浏览器测试的工具。
2. Mocha   前端自动化测试框架 chai sinon
3. Jest    Vue中继承的，简单，0配置。不能只能跑在浏览器，只能在node中操作dom，即jsdom


```
npm install --save-dev 
```
